# AIRAC Data Archive

Long-term version-controlled archive of input data for each AIRAC cycle, used by the [VFPC](https://github.com/VFPC) ecosystem.

## Structure

```
2603/
  2603_input.zip    # All input files for this cycle
  manifest.md       # Contents listing and any notes about this cycle
2604/
  2604_input.zip
  manifest.md
```

## What goes in each zip

- SRD CSV (main sheet exported from the NATS Excel workbook)
- SRD Notes CSV (notes sheet, if exported)
- UK sector file (.sct)
- in.json (parser configuration)
- AIP ENR 3.2 HTML
- AIP ENR 3.3 HTML
- aip_segments.json (generated by the AIP Parser)
- Any additional files relevant to this cycle (RAD spreadsheets, Eurocontrol data, etc.)

## Manifest

Each cycle's `manifest.md` documents what's in the zip and any notes about that cycle — unusual decisions, data quality issues, removed/added notes, etc. This is the human-readable audit trail.

## How files get here

Files are prepared by the [airac-data-fetcher](https://github.com/VFPC/airac-data-fetcher) tool and staged in this repo for manual review before committing.
